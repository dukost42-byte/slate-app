Сюда кладутся реальные файлы скриптов (.jsx для AE и т.д.).

Сейчас установка генерирует заглушку-скрипт на лету (см. src/main/install.js),
поэтому папка может быть пустой. Когда появятся настоящие скрипты, install.js
будет копировать их отсюда, а electron-builder упакует папку в ресурсы
приложения (extraResources в package.json).
