# Сборка установщиков Slate (Windows и Mac)

Приложение собирается через electron-builder. Готовые файлы появляются в папке `dist/`.

## Что получится

- **Windows:** `Slate-Setup-0.1.0.exe` — установщик (per-user, без прав администратора).
- **macOS:** `Slate-0.1.0-mac-universal.dmg` — образ для установки (Intel + Apple Silicon),
  плюс `.zip` с тем же приложением.

## Вариант 1 — собрать через GitHub Actions (рекомендуется)

Так получаются сразу обе платформы, даже если у тебя только один компьютер
(сборка идёт на Windows- и Mac-раннерах GitHub).

1. Залей код приложения (папку `slate`) в отдельный GitHub-репозиторий, например `slate-app`.
   Файл `.github/workflows/build-app.yml` уже внутри — это и есть сборка.
2. Создай релиз через тег. В терминале в папке репозитория:
   ```
   git tag v0.1.0
   git push origin v0.1.0
   ```
   (или на сайте: Releases → Draft a new release → создать тег `v0.1.0`).
3. Во вкладке **Actions** пойдёт сборка под обе ОС. По завершении установщики
   автоматически прикрепятся к релизу `v0.1.0` на вкладке **Releases**.
4. Дай бета-тестерам ссылку на страницу Releases — они скачают нужный файл.

Запустить сборку без тега (для проверки) можно вручную: Actions → Build app →
Run workflow. Тогда файлы лягут в artifacts прогона (качаются участниками репозитория).

## Вариант 2 — собрать локально

Нужен Node.js LTS. Mac-версию можно собрать **только на macOS**, Windows-версию —
на Windows.

- На macOS:
  ```
  npm install
  npm run dist:mac
  ```
- На Windows:
  ```
  npm install
  npm run dist:win
  ```

Результат — в папке `dist/`.

## Важно: приложение не подписано (для беты это нормально)

Подпись кода (Apple Developer, Windows-сертификат) стоит денег и не нужна на этапе
беты. Поэтому при первом запуске система предупредит, что разработчик неизвестен.
Объясни тестерам, как открыть:

- **macOS:** правый клик по приложению → «Открыть» → ещё раз «Открыть».
  Если пишет, что приложение «повреждено», выполнить в Терминале один раз:
  ```
  xattr -dr com.apple.quarantine /Applications/Slate.app
  ```
- **Windows:** при предупреждении SmartScreen нажать «Подробнее» → «Выполнить в любом случае».

## Что тестерам нужно после установки

1. Открыть Slate.
2. На вкладке After Effects или Premiere Pro нажать «Включить режим разработчика»
   (расширения CEP не подписаны и без этого Adobe их не покажет).
3. Нажать «Установить» у нужного плагина.
4. Перезапустить Adobe → Window → Extensions → нужная панель.

Приложение тянет плагины из публичного репозитория релизов, поэтому на время беты
он должен оставаться **Public** — иначе у тестеров скачивание не сработает.
